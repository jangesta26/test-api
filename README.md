
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Dockerfile Installation

```bash
$ docker build -t test-api .
#OR
$ docker build -t test-api -f Dockerfile.prod .

$ docker run  -p 3001:3001 nestjs-docker

$ 
```
